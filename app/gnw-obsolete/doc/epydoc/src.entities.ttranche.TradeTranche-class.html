create_model(self, prefix="")

source code 

The dynamics of the trade tranche are as follows:

  • Inflow/Outflow restricted to standard product's delivery period
    • Restricted inflow volume: in_flow[t] [MWh] == 0, for t not in DELIVERY_PERIOD
    • Restricted outflow volume: out_flow[t] [MWh] == 0, for t not in DELIVERY_PERIOD
  • Transacted (sold: trade_volume < 0; purchased: trade_volume > 0) gas volume in [MWh]:
    • Transacted volume balance equation: trade_volume[t] [MWh] == out_flow[t] [MWh] - in_flow[t] [MWh]
    • Delivery period volume equations: trade_volume[t] [MWh] == trade_volume[u] [MWh], for all t, u in DELIVERY_PERIOD must hold
  • Sales (trade_volume < 0) gas volume in [MWh]:
    • Non-negativity constraints: sales_position[t] [MWh] >= 0, for all t in DISPATCH_PERIODS (see variable construction in method create_lp_vars).
    • Sales volume constraints: sales_position[t] [MWh] >= -trade_volume[t] [MWh], for all t in DISPATCH_PERIODS
    • Minimal sales volume equation: sum(sales_position[t], t in DELIVERY_PERIOD) [MWh] >= CAPACITY_LIMITS[0]*sum(DISPATCH_PERIODS[t], t in DELIVERY_PERIOD) [MW*h] (--> semi-continuous: 0 or >= MIN)
    • Maximal delivery volume equation: sum(sales_position[t], t in DELIVERY_PERIOD) [MWh] <= CAPACITY_LIMITS[1]*sum(DISPATCH_PERIODS[t], t in DELIVERY_PERIOD) [MW*h]
    • Objective function component: sum(sales_position[t]*PRICE[0]*DISCOUNT_FACTOR[t], t in DELIVERY_PERIOD)
  • Purchase (trade_volume > 0) gas volume in [MWh]:
    • Non-negativity constraints: purch_position[t] [MWh] >= 0, for all t in DISPATCH_PERIODS (see variable construction in method create_lp_vars).
    • Purchase volume equation: purch_position[t] [MWh] >= trade_volume[t] [MWh], for all t in DISPATCH_PERIODS
    • Minimal purchase volume equation: sum(purch_position[t], t in DELIVERY_PERIOD) [MWh] >= CAPACITY_LIMITS[0]*sum(DISPATCH_PERIODS[t], t in DELIVERY_PERIOD) [MW*h] (--> semi-continuous: 0 or >= MIN)
    • Maximal purchase volume equation: sum(purch_position[t], t in DELIVERY_PERIOD) [MWh] <= CAPACITY_LIMITS[1]*sum(DISPATCH_PERIODS[t], t in DELIVERY_PERIOD) [MW*h]
    • Objective function component: (-1.0)*sum(purch_position[t]*PRICE[1]*DISCOUNT_FACTOR[t], t in DELIVERY_PERIOD)
